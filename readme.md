# 企业权限管理系统

## 项目技术：
#### 后端技术：Spring SpringMVC  mybatis  Spring Security 
#### 前端渲染：AdminLTE  jquery ajax  EL  JSTL JSP 

##项目模块介绍：
#### 用户管理：添加用户 修改用户密码 删除用户  登陆实现
#### 角色管理 ：添加角色 删除角色 修改角色
#### 资源权限管理：添加权限 修改权限 删除权限 
#### 权限控制：在源码层控制 
#### 记录操作日志 ， 以及产品管理和订单管理等
